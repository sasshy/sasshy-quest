◇ Googleカレンダー（読み込み専用）
Googleカレンダーの予定をQUESTの日/週/月ビューに読み込みます。SASSHY側からGoogleカレンダーには書き込みません。
v9では、同じGoogleアカウント内の複数カレンダー（プライベート用・仕事用・共有カレンダーなど）をまとめて読み込めます。Apps Script URLは /exec で終わるWebアプリURLを使ってください。
ICSを貼り付けて反映（URL読み込みが失敗する時）
Apps Script連携用コードを見る（繰り返し予定も反映しやすい）
下のコードをGoogle Apps Scriptに貼り、ウェブアプリとしてデプロイします。URLの末尾に ?token=自分で決めた文字列 を付けて、この欄に保存します。
標準ではアクセスできる全カレンダーを読み込みます。絞り込みたい場合はURL末尾に &calendarNames=プライベート,仕事 または &calendarIds=ID1,ID2 を追加できます。 
const TOKEN = 'change-this-secret';

// 標準では「自分がアクセスできる全カレンダー」を読み込みます。
// URLに &calendarNames=プライベート,仕事 を付けると名前で絞り込みできます。
// URLに &calendarIds=xxxxx,yyyyy を付けるとIDで絞り込みできます。
function doGet(e) {
  if (e.parameter.token !== TOKEN) {
    return ContentService.createTextOutput('forbidden');
  }

  const callback = e.parameter.callback || '';
  const from = e.parameter.from ? new Date(e.parameter.from) : new Date();
  const to = e.parameter.to ? new Date(e.parameter.to) : new Date(Date.now() + 30*24*60*60*1000);

  const idFilter = splitParam_(e.parameter.calendarIds);
  const nameFilter = splitParam_(e.parameter.calendarNames);

  const calendars = CalendarApp.getAllCalendars().filter(cal => {
    const id = cal.getId();
    const name = cal.getName();
    if (idFilter.length && !idFilter.includes(id)) return false;
    if (nameFilter.length && !nameFilter.includes(name)) return false;
    return true;
  });

  const events = [];
  const calendarSummaries = [];

  calendars.forEach(cal => {
    const calendarId = cal.getId();
    const calendarName = cal.getName();
    const calendarColor = safe_(() => cal.getColor(), '');

    const calEvents = cal.getEvents(from, to);
    calendarSummaries.push({
      id: calendarId,
      name: calendarName,
      color: calendarColor,
      count: calEvents.length
    });

    calEvents.forEach(ev => {
      events.push({
        id: calendarId + '::' + ev.getId(),
        eventId: ev.getId(),
        calendarId,
        calendarName,
        calendarColor,
        title: ev.getTitle(),
        start: ev.getStartTime().toISOString(),
        end: ev.getEndTime().toISOString(),
        allDay: ev.isAllDayEvent(),
        location: ev.getLocation() || '',
        description: ev.getDescription() || ''
      });
    });
  });

  const json = JSON.stringify({
    events,
    calendars: calendarSummaries,
    updatedAt: new Date().toISOString()
  });

  if (callback) {
    return ContentService.createTextOutput(callback + '(' + json + ')')
      .setMimeType(ContentService.MimeType.JAVASCRIPT);
  }

  return ContentService.createTextOutput(json)
    .setMimeType(ContentService.MimeType.JSON);
}

function splitParam_(value) {
  return String(value || '')
    .split(',')
    .map(s => s.trim())
    .filter(Boolean);
}

function safe_(fn, fallback) {
  try {
    return fn();
  } catch (e) {
    return fallback;
  }
}
